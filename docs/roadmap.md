# Roadmap

## Phase 1: Visual MVP
- Node/edge editing with planes and palettes.
- Workspace persistence and version snapshots.
- Plan diff generation and review UI.

## Phase 2: Provider adapters
- Map generic nodes to cloud/bare-metal providers.
- Integrate OpenTofu plan/apply as an adapter.
- Cost estimation hooks per provider.

## Phase 3: Drift + collaboration
- Live drift checks and reconciliation flows.
- Multi-user roles, comments, and approvals.
- Plan audit history and exportable reports.
